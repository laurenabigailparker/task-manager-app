# Task Manager App

A modern task management application built with HTML, CSS, and JavaScript. This app allows users to create, organize, and manage tasks with categories and completion tracking.

## Features
- Add and delete tasks
- Categorize tasks (Home, School, Work, etc.)
- Mark tasks as completed
- Dynamic task rendering
- Responsive design
- Clean and modern UI

## Tech Stack
- HTML
- CSS (Flexbox + Grid)
- JavaScript (DOM Manipulation)

## How It Works
Tasks are stored in an array of objects. Each task contains:
- text
- category
- completed status

The UI is updated dynamically using a render function that rebuilds the task list whenever changes occur.
